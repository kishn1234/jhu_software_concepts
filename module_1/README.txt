Module 1 Assignment: Personal Website

Project Description:
This project is a personal student portfolio website built using
Python, Flask, HTML, CSS, and GitHub. The website includes
an About page, Publications page, and Contact page.

How to Run the Project:

1. Open terminal
2. Navigate to the module_1 folder
3. Run the following command:

python3 run.py

4. Open a browser and go to:

http://localhost:8080

Required Packages:
Flask
